# 🎮 HTTP Status Game

An interactive browser-based game built with Phaser 3 to help developers and students learn HTTP status codes in a fun and engaging way.

---

## 📌 Overview
HTTP Status Game transforms traditional learning into an interactive experience. Players navigate through challenges that simulate real-world HTTP responses such as **200 OK**, **404 Not Found**, and **500 Internal Server Error**.

---

## ✨ Features
- 🎮 Interactive gameplay powered by Phaser 3  
- 📚 Learn common HTTP status codes through practice  
- 🧩 Multiple levels *(with more coming soon)*  
- ⚡ Fast, lightweight, and browser-friendly  
- 🌍 Deployed and accessible online via Vercel  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, JavaScript  
- **Game Engine:** Phaser 3  
- **Deployment:** Vercel  
- **Version Control:** Git & GitHub
