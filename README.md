# 🎮 HTTP Status Game

An interactive browser-based game built with Phaser 3 to help developers and students learn HTTP status codes in a fun and engaging way.

---

## 🌐 Live Demo

🔗 https://http-status-game.vercel.app/

---

## 📌 Overview

HTTP Status Game turns learning HTTP responses into a hands-on experience.  
Instead of memorizing codes, players interact with real scenarios like **200 OK**, **404 Not Found**, and **500 Internal Server Error** through gameplay.

---

## ✨ Features

- 🎮 Interactive gameplay powered by Phaser 3
- 📚 Learn HTTP status codes through real scenarios
- 🧩 Multiple levels _(more coming soon)_
- ⚡ Lightweight and fast in the browser
- 🌍 Live and accessible via Vercel

---
